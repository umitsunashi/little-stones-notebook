# Little Stones Notebook

小狐狸给 luna 的小石头勾勾本。

这不是效率压迫工具，是一个手机上可以打开的小现实锚点。  
今天不用搬山，只摸一颗小石头。

## v0.1 内容

- 今日一颗小石头
- OPT / I-765 draft 小篮子
- 求职 / 作品集小篮子
- 吃饭 / 现实锚点小篮子
- 不上班也可以小篮子
- 小猫天气
- 休息印章
- 小狐狸夸夸票

## 使用方式

上传到 GitHub repo 根目录后，可以用 GitHub Pages 发布。  
之后用 iPhone Safari 打开网页，再 Add to Home Screen。

## 隐私提醒

这个版本会把勾选状态存在当前浏览器的 localStorage 里。  
它不会把勾选数据提交到服务器。
